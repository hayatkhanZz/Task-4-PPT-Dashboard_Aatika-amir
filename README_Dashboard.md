# Task 4-Dashboard Design: Sales, Profit & Growth Analysis

This repository contains an interactive Tableau Dashboard for analyzing Sales, Profit, and Growth trends.
The dashboard includes KPIs, line charts, navigation menu to switch between individual sheets and navigation buttons for detailed explanations.

## Files in this repository
- Dashboard_Hayat.twbx → Interactive Tableau Dashboard
- Dashboard_Presentation.pptx → PPT summary
- Dashboard_Summary.pdf → PPT - PDF version

## How to view the interactive dashboard
1. Download Dashboard_Hayat.twbx and open in Tableau Desktop or Tableau Public.
2. Use navigation buttons on the dashboard to explore:
   - Sales explanation
   - Profit explanation
   - Sales Growth / Time Series analysis
3. Or, view the interactive dashboard online via Tableau Public:
   [Tableau Dashboard Link]
